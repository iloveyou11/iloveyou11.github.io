  • 计算机

基本流程

1、递归+记忆化——递推
2、状态的定义
3、状态转移方程
4、最优子结构

比较概念

DP vs 回溯（递归） vs 贪心
DP——记录局部最优子结构/多种记录值
回溯（递归）——重复计算
贪心——永远局部最优

实战演练

0、二维矩阵计算走法种类（存在障碍）
1、爬楼梯（每次只能走一阶或两阶）

2、三角形最小路径之和(m 层)
状态定义：DP[i][j]——从底部走到[i][j]的所有路径最小和

3、乘积最大子序列 (2 3 -2 4)
如果不用连续
如果连续
1）暴力求解？

4、股票买卖问题7, 1, 3, 5, 2, 4, 1

5 最长上升子序列(位置不变)
1)不用连续
方法 1:暴力求解
方法 2:动态规划 O(N^2)
状态定义:DP[i] 到第 i 个数(选上 i 元素)的最优子结构
结果:DP[0]到 DP[n-1]的最大值
状态转移方程:DP[i]=max(DP[j])+1 (j 从 0 到 i-1,且 a[j]<a[i])
方法 3:二分法 O(NlogN)

2)要求连续(更加容易)
